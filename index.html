Tải Game

Tải file game bên dưới.

Cách mở game

  1. Cài Code::Blocks.
  2. Mở file .cbp hoặc .cpp.
  3. Nhấn F9 để Build & Run.

Cách chơi

  • Story: đánh quái.
  • Hero Shop: mua nhân vật.
  • Hero Chest: mở rương.
  • Shard Exchange: đổi mảnh.
  • Upgrade: nâng cấp nhân vật.
  • Redeem: nhập code.
